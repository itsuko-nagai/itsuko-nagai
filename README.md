## About Me

I work at the intersection of robotics, embedded systems, signal processing, and machine learning. My focus is on designing reliable, optimized, and real-time intelligent systems — bridging hardware, firmware, and deep learning.

- **Building:** Real-time intelligent robotics, embedded ML systems  
- **Interests:** Robotics frameworks, control systems, autonomous systems, ML for hardware  
- **Learning:** Vision-based deep learning, reinforcement learning, advanced embedded hardware  
- **Expertise:** Robotics development, hardware–software integration, ML deployment, signal processing
